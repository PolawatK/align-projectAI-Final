# 📚 README — วิธีปรับ RAG Knowledge Base

ไฟล์นี้อธิบายวิธีเพิ่ม/แก้ไข/ลบความรู้ใน Chatbot ของ ALIGN

---

## ไฟล์ที่ต้องแก้

```
backend/src/data/knowledge.js
```

**แค่ไฟล์เดียว** — ไม่ต้องแตะโค้ดอื่นเลย

---

## โครงสร้าง Entry

```js
{
  id: 'xx_001',          // unique ID ห้ามซ้ำ
  category: 'string',   // หมวดหมู่ (ดูรายการด้านล่าง)
  tags: ['คำ1', 'คำ2'], // keywords สำหรับค้นหา (ยิ่งมากยิ่งดี)
  title: 'หัวข้อ',      // ชื่อสั้นๆ
  content: `เนื้อหา`    // ความรู้ละเอียด (ยิ่งละเอียดยิ่งดี)
}
```

---

## Categories ที่มีอยู่

| Category | ความหมาย |
|----------|---------|
| `office_syndrome` | สาเหตุ อาการ การป้องกัน Office Syndrome |
| `posture_correction` | การแก้ท่าทาง หลังค่อม คอยื่น |
| `exercises` | ท่าบริหาร ยืดกล้ามเนื้อ |
| `ergonomics` | การจัดโต๊ะ เก้าอี้ อุปกรณ์ |
| `pain_relief` | บรรเทาอาการปวด |
| `lifestyle` | การนอน การใช้โทรศัพท์ |
| `when_to_see_doctor` | เมื่อไหร่ควรพบแพทย์ |

เพิ่ม category ใหม่ได้เลย แค่ตั้งชื่อ string ใหม่

## ตัวอย่าง: เพิ่มความรู้ใหม่

```js
// เพิ่มต่อจาก entry สุดท้ายใน knowledgeBase array
{
  id: 'ex_004',                          // เพิ่มเลขต่อ
  category: 'exercises',
  tags: ['yoga', 'โยคะ', 'ยืดหยุ่น', 'บริหาร', 'ท่า'],
  title: 'โยคะท่าพื้นฐานสำหรับ Office Syndrome',
  content: `ท่าโยคะที่ช่วยแก้ปัญหาจากการนั่งทำงาน:

1. Cat-Cow (แมว-วัว)
   - คุกเข่า มือวางบนพื้น
   - หายใจเข้า: แอ่นหลัง ยกหัวขึ้น (Cow)
   - หายใจออก: โค้งหลังขึ้น ก้มหัวลง (Cat)
   - ทำ 10 รอบ ช่วยคลายหลังและคอ

2. Child's Pose (ท่าเด็ก)
   - นั่งทับส้นเท้า ก้มลงเหยียดแขนไปข้างหน้า
   - ค้าง 30-60 วินาที หายใจลึกๆ
   - ยืด thoracic spine และสะโพก

3. Seated Spinal Twist (บิดลำตัว)
   - นั่งขัดสมาธิ มือซ้ายบนเข่าขวา
   - บิดลำตัวไปขวา มองข้ามไหล่
   - ค้าง 30 วินาที สลับข้าง`
}
```

## ตัวอย่าง: แก้ไขข้อมูลที่มีอยู่

ค้นหา `id: 'pc_002'` แล้วแก้ `content` ได้เลย:

```js
{
  id: 'pc_002',
  category: 'posture_correction',
  tags: ['หลังค่อม', 'kyphosis', ...],
  title: 'วิธีแก้หลังค่อม (Kyphosis)',
  content: `... แก้เนื้อหาตรงนี้ ...`
}
```

## ตัวอย่าง: ลบ Entry

ลบ object ทั้ง block ออกจาก array ได้เลย เช่น ลบ `id: 'ls_002'` ออก

## Tips การเขียน Content ที่ดี

### ✅ ดี
```
ท่า Chin Tuck:
1. นั่งตัวตรง
2. ดึงคางกลับ ไม่ก้มหัว
3. ค้าง 5 วินาที ทำ 10 ครั้ง
4. ทำทุก 1 ชั่วโมง
```

### ❌ ไม่ดี
```
ให้ทำ chin tuck โดยดึงคางกลับและทำบ่อยๆ
```

**หลักการ:**
- ใส่ตัวเลขที่ชัดเจน (วินาที, ครั้ง, เซ็ต, วัน)
- แบ่งเป็นข้อๆ ง่ายต่อการอ่าน
- ระบุคำเตือนถ้ามี
- ใส่ทั้งภาษาไทยและ English สำหรับ tags

## Tips การตั้ง Tags

Tags ใช้สำหรับจับคู่กับคำถาม ยิ่งมาก ยิ่งถูกเรียกใช้บ่อย

```js
tags: [
  'หลังค่อม',        // ภาษาไทย
  'kyphosis',        // ภาษาอังกฤษ  
  'หลังโก่ง',        // คำพ้องความหมาย
  'แก้หลัง',         // วิธีที่คนถาม
  'upper back',      // English alternative
  'posture'          // broader term
]
```

## วิธี Deploy หลังแก้ไข

เนื่องจาก knowledge base เป็น JavaScript file ธรรมดา แค่ commit และ push ขึ้น GitHub:

```bash
git add backend/src/data/knowledge.js
git commit -m "update: เพิ่มความรู้เรื่อง yoga"
git push
```

Render จะ redeploy อัตโนมัติภายใน 2-3 นาที ✅

## ระบบ Retrieval ทำงานอย่างไร

เมื่อผู้ใช้ถามคำถาม ระบบจะ:

1. **ค้นหา** entries ที่ tags/title/content ตรงกับคำถาม
2. **คำนวณ score** แต่ละ entry (tag match = 3pt, title = 2pt, content = 0.5pt)
3. **เลือก top 3** entries ที่คะแนนสูงสุด
4. **ส่งให้ Gemini** พร้อมกับคำถาม เพื่อสร้างคำตอบ

ยิ่ง tags ตรงกับคำถามมาก ยิ่งถูกเลือก → ตอบได้แม่นยำขึ้น

## ตัวอย่าง: เพิ่มหมวดใหม่ทั้งหมด

```js
// หมวดใหม่: โรคอ้วน/น้ำหนักกับท่าทาง
{
  id: 'wt_001',
  category: 'weight_posture',  // category ใหม่
  tags: ['น้ำหนัก', 'อ้วน', 'BMI', 'หลัง', 'เท้า', 'ข้อเข่า'],
  title: 'น้ำหนักตัวกับสุขภาพกระดูกสันหลัง',
  content: `น้ำหนักตัวมีผลโดยตรงต่อกระดูกสันหลัง...`
}
```

ไม่ต้องลงทะเบียน category ใหม่ที่ไหนทั้งนั้น แค่ตั้งชื่อใน `category` field ได้เลย

## Q&A

**Q: เพิ่มกี่ entry ได้?**  
A: ไม่จำกัด แต่แนะนำไม่เกิน 50 entries เพื่อ performance

**Q: เนื้อหาภาษาอังกฤษได้ไหม?**  
A: ได้ แต่ถ้าอยากให้ bot ตอบภาษาไทย ควรใส่ content เป็นไทยหลัก

**Q: ถ้าถามนอกขอบเขตใน KB จะเกิดอะไร?**  
A: Gemini จะใช้ความรู้ทั่วไป แต่ยังถูกจำกัดโดย system prompt ให้อยู่ในเรื่องท่าทาง

**Q: ปรับบุคลิก bot ได้ไหม?**  
A: ได้ แก้ที่ `SYSTEM_PROMPT` ใน `backend/src/routes/chat.js`
